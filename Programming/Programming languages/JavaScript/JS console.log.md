# JS console.log

Используется для выода в консоль текста. 

```js
console.clear(); //очищает консоль
console.log('t:', t); //выводит текст и переменную

console.info("text") //не отличается от обычного
console.warn("Warn") 
console.error("Error") //красным

console.group();
...
...
console.groupEnd();
```
